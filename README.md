# rock-paper-scissors

This is a Rock Paper Scissors game played in the browser console between the computer random choice and the human user input via prompt()

## How to run

Open `index.html` in a browser and check the console as you play — the game will prompt you for a choice 5 times and then declare a winner.

## skills practiced

- Exercising problem solving and breaking down the assignment into a smaller solvable problems.
- Creating and working with Functions.
- Writing and simplifying if/else statements.
- Practicing debugging when faced with Errors.

## what i learned

- working with Math.random() and making it choose from a specific number of options
